Codecademy is a online learning platform that teaches programing. From languages to frameworks, anyone can learn something new at codecademy. I started a certificate course "web development", where I learned terminology of we development as well as hmtl, css and javascript. I am currently still pursuing the certificate but I am on a good way to get it. Learn more

FREECODECAMP JAVASCRIPT BLOCKCHAIN COURSE

This is the first real full web3 learning tutorial that set me up with all the needed techniques to write smart contracts and deploy them. In this course I learned solidity smart contract programming, javascript for web3 development, nextjs framework, tailwindcss for css, html, ipfs. I have also learned core blockchain concepts such as ERC20, ERC721, DAO, DeFi and much more. After this course I could confidently write and deploy smart contracts, it is the best course regarding web3 and programming. Learn more

CRYPTO ZOMBIES

CryptoZombies is a very interesting online learning platform focusing on solidity development. The courses are very well structured and while you are learning you are actually building a NFT game. I learned a lot about connecting smart contracts to each other, ERC20, ERC712, and a little frontend web3 development in react.Learn more

MORALIS ACADEMY
